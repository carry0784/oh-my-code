# Promotion Review Template
## 승격 심사 템플릿 v1.0

**Document Path:** `docs/operations/promotion_review_template.md`
**Parent Authority:** `Operating Constitution v1.0`
**Status:** ACTIVE
**Scope:** 승격 심사 표준 양식·필수 포함 항목·판정 등급 고정
**역참조:** 제39조, 제45조

---

## 제약 선언

- 본 문서는 상위 헌법의 구현 세부화 문서이다.
- 본 문서는 새 권한을 추가하지 않는다.
- 본 문서는 자동 실행 권한을 부여하지 않는다.
- 본 문서는 상위 헌법과 충돌해서는 안 된다.
- 심사 완료는 자동 승격을 의미하지 않는다.

---

## 1. 사용 범위

본 템플릿은 다음 판단에 사용한다.

- 실거래 안정화 단계 전환
- 자본 확대 판단

---

## 2. 판정 등급

| 등급 | 의미 |
|------|------|
| `APPROVED` | 승격 가능 (자동 실행 아님, 별도 운영 조치 필요) |
| `APPROVED_WITH_LIMITS` | 조건부 승격 (제한 항목 명시 의무) |
| `DEFERRED` | 보류 (추가 관찰 후 재심사) |
| `REJECTED` | 거부 (승격 불가, 사유 기록 필수) |

**APPROVED_WITH_LIMITS 사용 시 반드시 제한 항목을 명시해야 한다.**
제한 필드 생략은 허용되지 않는다.

---

## 3. 필수 포함 항목 (제45조)

모든 승격 심사는 다음 6개 항목을 반드시 포함해야 한다.

| 항목 | 설명 |
|------|------|
| 최근 운영 기간 | 심사 대상 기간 |
| CRITICAL 수 | 기간 내 CRITICAL 발생 건수 |
| WARNING 추세 | WARNING 발생 추세 (증가/감소/안정) |
| 정합성 위반 수 | 기간 내 정합성 위반 건수 |
| 수동 개입 빈도 | 운영자 수동 개입 횟수 |
| 자본 확대 허용 여부 | 자본 확대 가능 판단 |

## 5. 금지 사항

- 심사 템플릿에 자동 승격 실행 권한 부여 금지
- 자본 확대 자동 실행 권한 부여 금지
- LOCKDOWN 자동 해제 권한 부여 금지
- 미완료 evidence를 완료로 간주하는 표현 금지
- 제한 승인에서 제한 필드 생략 허용 금지

---

## 6. APPROVED ≠ 자동 실행

심사에서 APPROVED 판정을 받더라도, 이는 승격을 자동으로 실행하는 것이 아니다.
별도의 운영 조치를 통해 승격을 수행해야 한다.
